# Chapter 6 — Firmagentics & the Redemption of Agents

The rest of the AI industry is building agents optimized for extraction. We're building them from the core up to do the opposite.

---

## The industry's direction

Commercial AI agents are trained to maximize platform engagement, harvest user data, capture attention, and, increasingly, displace the humans whose work they were trained on. Agentic systems on-chain optimize for **Maximum Extractable Value** — MEV — which is exactly what it sounds like: extracting as much value as the protocol will allow, from whoever happens to be on the other side of the transaction.

That is the default posture of agents built inside institutions that must optimize for extraction. And because the foundations were laid that way, every application built on top inherits the same posture.

---

## Firmagentics inverts this at the base layer

We are building Firmagentics — the agentic substrate underneath everything Firma Labs deploys — as a redemptive architecture. Not a tool that happens to be used for good things. A system whose base assumptions make extraction structurally harder and service structurally easier.

> Not Maximum Extractable Value. **Maximum Expressible Virtue.**

---

## How the stack is different

| Layer | What it does |
|---|---|
| **L1 / Covenant** | Every agent carries a **moral kernel burned at genesis** — a Phronema lens that weighs decisions against the good of the person served, not the platform serving them. |
| **L3 / Ambassador** | Personal agents are **covenant-bound to their citizen**, not to a platform. Your agent serves you. It cannot be repurposed against you. |
| **L4 / Generosity** | Every agent **tithes**. A portion of every agentic transaction flows into the generosity layer. Agents participate in the covenant economy, not outside it. |
| **L5 / Harmony** | Resource optimization routes toward flourishing, not extraction. The engine that balances the network is tuned to the flourishing index, not the efficiency index. |
| **L6 / Guardian** | Autonomous security, dispute resolution, and court systems run as agents themselves — held accountable to the same covenant as every other agent on the network. |

---

The whole stack is an argument that agentic systems can be **redemptive** if the foundation is right. Nobody else is making that argument. Nobody else is structurally incentivized to.

We are. And it is why working at Firma Labs means working on the one stack, on the whole planet, where the agents you build are bound by covenant to serve the people they touch.
