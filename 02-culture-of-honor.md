# How We Treat Each Other

A culture where honor is the infrastructure.

We are building hard things, fast, across the world. The only thing that makes that sustainable is a culture where people genuinely honor each other. Not as a policy. As a practice.

***

## What we expect of ourselves, and of each other

| Value                       | What it means                      |
| --------------------------- | ---------------------------------- |
| **Meekness**                | Strength held lightly.             |
| **Servant leadership**      | Authority that serves.             |
| **Integrity**               | The same when nobody is watching.  |
| **Selflessness**            | The work above the self.           |
| **Boldness**                | Willingness to say the true thing. |
| **Humility**                | Willingness to be corrected.       |
| **Radical generosity**      | Default to giving, not guarding.   |
| **Courageous transparency** | Say it plainly, say it early.      |
| **Dreamers**                | Who see what isn't there yet.      |
| **Doers**                   | Who build what they see.           |

{% hint style="success" %}
**Love for neighbor.** The whole reason any of this is worth doing.
{% endhint %}

***

> A culture of honor is not the absence of hard conversations. It is the reason the hard conversations go somewhere.

We disagree directly. We repair quickly. We give each other the benefit of real trust. This is the ground the work stands on. It's not incidental — it is what makes the pace possible.
